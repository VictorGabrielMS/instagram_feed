## :octocat:Clone this repo

1. Clone :

```sh
  $ git clone https://github.com/msvictor/uiclone_instagram.git
```

2. Go to Folder:

```sh
  $ cd uiclone_instagram
```

3. Execute:

```sh
  $ npm install
  $ npm run android
  # or
  $ npm run ios
```
